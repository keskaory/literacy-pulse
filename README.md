# literacy-pulse

Exploring the decline in American reading proficiency and how shifting media habits might be connected.

Built with Python, DuckDB, Streamlit, and Plotly.

## Data Sources
- **NAEP** — national reading scores for grades 4, 8, and 12 (1998–2024)
- **Pew Research** — media consumption trends over time (in progress)
